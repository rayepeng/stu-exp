    Too cold for steg¶

    Given a file final.txt. When I try to open it, the content is a Sherlock Holmes Novel. Based on the title Too Cold, I guess that this challenge will be related to stegsnow tools. In order to extract the hidden data, I need to find the password. I try to look it on the file and the password was store on there.

    The command to extract it:

    stegsnow -C -p “d4rkc0de-IIITD” final.txt

    后来才得知这是一种文本隐写，将信息隐藏在文本中

    stegsnow

    -m指定要隐藏的信息，source.txt是原来的文件，readme.txt是隐藏了信息的文件

    1
    stegsnow -C -m 'We attack at dawn with Legos!' source.txt readme.txt

    提取信息

    1
    stegsnow -C readme.txt

    将文件隐藏到另一个文件中

    1
    stegsnow -C -f hidden_message.txt source.txt readme.txt

    提取文件

    1
    stegsnow -C readme.txt

    加入密码保护

    1
    stegsnow -C -p "hello12345" -f hidden_message.txt source.txt readme.txt

    解密

    1
    stegsnow -C -p 'hello12345' readme.txt
